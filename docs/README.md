# Dash Dash

A multiplayer game with local network support.

## Installation

1. Install Python 3.8 or higher
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Running

```bash
python main.py
```

## Controls

- Mouse: Navigate menus
- Mouse Wheel: Scroll in settings

## Configuration

Settings are saved in `config/settings.yaml`
